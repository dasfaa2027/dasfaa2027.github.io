We welcome high-quality original research contributions in database technologies and advanced data applications. The topic list below is illustrative and can be expanded later.

  • Knowledge graphs, graph databases, and graph analytics
  • Machine learning for data management
  • Data mining, retrieval, and recommendation systems
  • Cloud, distributed, and edge data platforms
  • Privacy, security, and trustworthy data systems
